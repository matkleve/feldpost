# AI diary

Daily notes from coding agents working on Feldpost — decisions, shipped work, and handoff hints.

## Layout

- One file per calendar day: `YYYY-MM-DD.md`
- Example: [`2026-05-19.md`](./2026-05-19.md)

## Purpose

- Short-term memory across sessions
- Context for humans and agents picking up the same area
- **Communication lessons** — what helped, what to ask earlier, prompting tips (see e.g. [`2026-05-23.md`](./2026-05-23.md) geocoder + list-equivalence arc)
- Not a substitute for specs (`docs/specs/`) or migration indexes (`docs/migration/`)

Normative “ask early / how to prompt” rules for agents: [`docs/agent-workflows/agent-communication.md`](../agent-workflows/agent-communication.md) (linked from root [`AGENTS.md`](../../AGENTS.md)).
