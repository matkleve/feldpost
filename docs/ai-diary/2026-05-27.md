# 2026-05-27

## Sort dropdown empty list (Media toolbar)

**Symptom:** Sort panel showed search + divider only; no sort properties (Date captured, City, …).

**Cause (two bugs):**
1. `:host.workspace-toolbar-menu-panel { min-height: 0 }` collapsed the scroll band (fixed: removed override).
2. **Nested content projection:** `app-menu-panel-scroll-region` wrapped `ng-content` inside `app-standard-dropdown`, so `[dropdown-items]` never reached the DOM — DevTools showed an empty `.standard-dropdown__items` div.

**Fix:** Restore a single projection site in `standard-dropdown.component.html` (`<div [class]="itemsHostClass()">` + `ng-content`), using `menuPanelScrollHostClasses` / `menuPanelScrollOverflowClasses` for `scrollMode`.

**Verify:** Open `/media` → Sort → list shows built-in properties; Grouping unchanged.
