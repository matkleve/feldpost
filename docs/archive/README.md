# Archive

## Policy

- Archived code snapshots are stored directly under the archive root.
- Use date/refactor folder naming for snapshots (pattern: `YYYY-MM-DD-refactor-name`).
- Do not use an extra `code-legacy/` parent folder.
- Keep TypeScript snapshot suffix `.legacy.ts`.
- Never hard-delete replaced code before creating archive snapshots.
